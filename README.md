# The Conspiracy Transport Suite (CTS)

A seven-layer reference model for unverified belief propagation online,
laid out to the Internet-Draft spec.

`draft-the-midge-conspiracy-stack-00.txt` is the document. Open it in a
monospace viewer. 72-column body, RFC 2119 requirements language, a
32-bit packet header diagram, BCP 78/79 boilerplate, form-feed page
breaks, the lot.

SMTP frames the meme. UDP transports it fire-and-forget, no proof and no
shame, because acknowledgement is for sheep. BGP route-hijacks it out of
one fringe forum and onto every timeline downstream. DNS resolves your
vague unease to an actual Facebook group. TLS terminates its trust chain
at a guy in a shed who signed the certificate with a biro.

The R and A bits in the SMTP header (Red Circle, Arrow) both MUST be set.

Read the BGP section. It's the one that's genuinely close to legit if you
think about it.

Cites RFC 1925.
